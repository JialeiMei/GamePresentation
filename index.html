Purpose. Fun Math helps your child learn math.

Children should have the fundamentals mastered at an early age. Fun Math game is exposed to almost every child and we aim to use this approach to aid children in learning math. Utilize video games to keep children entertained while learning at the same time. Fun math is a quiz-form game that will enhance a childâ€™s math fundamentals and capabilities.

Game Features Travel from one map to another.

The user will use WASD keys to control the character to move up, left, down and right. There are different kind of house and each indicates the different math operation equation. There are addition, subtraction and multiplication.

Quiz Gamestyle Test yourself in random generated question.

Once the user enters the house, the game asks you to choose the level of the equation you want to answer. Then the equation appear, if you get it right, you get the score.
